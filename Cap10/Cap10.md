**Microdata (Dive Into HTML5)**.  

Este capítulo explica **Microdata**, una forma de añadir información semántica a HTML para que las máquinas (buscadores, crawlers, asistentes, etc.) entiendan mejor el contenido de una página web, sin afectar cómo la ven los humanos.

**¿Qué es Microdata?**.  
Microdata es una especificación de HTML5 que permite **anotar datos estructurados** dentro del HTML usando atributos especiales como:
- `itemscope`
- `itemtype`
- `itemprop`

Su objetivo es describir **qué es** el contenido, no solo cómo se ve.

**El modelo de datos de Microdata**.  
Microdata se basa en:
- **Items** → objetos del mundo real (personas, eventos, productos, etc.)
- **Properties** → características de esos objetos (nombre, fecha, precio…)

Los items pueden:
- Tener múltiples propiedades
- Estar anidados dentro de otros items
- Compartir vocabularios comunes

**Marcando personas**.  
Se muestra cómo representar una persona usando Microdata, por ejemplo:
- Nombre
- Foto
- Correo
- URL personal

Esto ayuda a que buscadores entiendan que esos datos pertenecen a **una persona real**, no solo texto suelto.

**Google Rich Snippets**.  
Google usa Microdata para mostrar resultados enriquecidos (**Rich Snippets**), como:
- Información adicional
- Detalles visuales
- Datos destacados en los resultados de búsqueda

Microdata **no garantiza** Rich Snippets, pero los hace posibles.

**Marcando organizaciones**.  
También se puede describir:
- Empresas
- Instituciones
- Sitios web

Indicando datos como:
- Nombre
- Logo
- Dirección
- Página oficial

**Marcando eventos**.  
Microdata permite estructurar eventos con:
- Nombre del evento
- Fecha y hora
- Lugar
- Descripción

Esto facilita que buscadores entiendan cuándo y dónde ocurre algo.

**Reseñas y valoraciones**.  
Se explica cómo marcar:
- Opiniones
- Calificaciones
- Reviews

Esto es lo que permite que Google muestre ⭐⭐⭐⭐ en los resultados.

**Idea clave del capítulo**.  
Microdata:
- No cambia el diseño de la página
- Hace el contenido **más comprensible para las máquinas**
- Mejora la interoperabilidad de la web
- Es una base para SEO moderno y web semántica

**Conclusión**.  
Microdata es un puente entre:
- HTML tradicional (para humanos)
- Datos estructurados (para máquinas)

Es una herramienta poderosa para hacer la web más inteligente sin complicarla.
